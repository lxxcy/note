## [mysql服务](#目录)
启动与停止：
注意需要在管理员模式下启动windows命令行
- 启动
  `net start mysql80`
- 停止
  `net stop mysql80`

客户端连接
注意前提是mysql服务启动
- 方法一：MySQL提供的客户端命令行工具
- 方法二：利用命令行工具执行`mysql [-h 127.0.0.1] [-p 3306] -u root -p`
  
## [mysql数据模型](#目录)
数据库分类：
- 关系型数据库  
  概念：建立在关系模型顶基础上，由多张相互连接的**二维表**组成的数据库  
  特点：
  - 使用表存储数据，格式统一，便于维护
  - 使用SQL语言操作，标准统一，使用方便
- 非关系型数据库
